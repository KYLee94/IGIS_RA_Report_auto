IOTA Seoul Data Package v0.10

IOTA Seoul 개발관리 Workspace 및 LFC 화면 제작/데이터 검수용 산출물입니다. 기준일은 2026-04-30입니다.

바로 다운로드

전체 패키지 ZIP엑셀 3종, build report, README.md를 한 번에 받습니다. README.md각 파일의 목적과 확인 순서를 설명한 안내문입니다. IFPDP HandOff v0.10제작 담당자가 먼저 볼 요약형 엑셀입니다. DB for UI v0.10UI 화면 제작에 바로 쓰기 위한 정제 DB입니다. Master DB v0.10스키마와 검수용 메타를 유지한 마스터 DB입니다. Build Report원천 파일 색인, LFC 보강, 구조 검증 결과입니다.

권장 확인 순서

  1. 먼저 IFPDP HandOff v0.10의 요약 시트를 봅니다.
  2. LFC 화면 제작값은 v0.10_LFC_DATA와 계획_시나리오값을 봅니다.
  3. 충돌값과 미충전값은 추가 원천 확인 전까지 화면 확정값으로 쓰지 않습니다.
